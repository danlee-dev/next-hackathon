# Test Fixtures

테스트할 때 `/pitch/new` 화면에 그대로 복사·붙여넣기할 수 있는 IR 발표 dataset.
난이도·도메인 별 4종.

| 파일 | 도메인 | 난이도 | 의도된 약점 |
|------|--------|-------|-------------|
| 01-strong-saas.md | B2B SaaS | A급 | 거의 없음 (S/A 등급 나와야 정상) |
| 02-weak-fintech.md | 핀테크 | C급 | 시장 규모 모호, 공허한 표현 다수 |
| 03-medical-ai.md | 의료 AI | B급 | 규제 리스크 누락, traction 약함 |
| 04-edtech-aggressive.md | 에듀테크 | F급 | 과장 과대, '경쟁자 없음' 단정 |

## 사용법

각 MD 파일은 다음 3개 섹션:

1. **세션 이름** — title 입력란에 박을 한 줄
2. **발표 대본** — script textarea에 박을 본문
3. **심사 기준** — judging_criteria textarea에 박을 평가 항목
4. **(선택) IR 덱 텍스트** — PDF 업로드 대신 텍스트로 동등 효과 (다음 라운드에 PDF 변환)

## 평가 검증

각 데이터셋은 *예상 결과 범위*도 함께 명시. 실제 LLM 출력이 그 범위 안에 들어오면
프롬프트가 잘 작동하는 신호. 너무 후하거나 너무 박하면 prompt 튜닝 필요.

## 검증 체크리스트

- [ ] 01: trust_grade S 또는 A. 김팩트가 traction/숫자 인정.
- [ ] 02: trust_grade C. 박독설이 '혁신적인' 등 인용. 김팩트가 시장 규모 출처 요구.
- [ ] 03: trust_grade B. 이공감이 진정성 인정 + 박독설이 규제 디테일 짚음.
- [ ] 04: trust_grade F. 김팩트가 '경쟁자 없음'을 시장 없음과 동일시. 박독설 인용 폭격.
