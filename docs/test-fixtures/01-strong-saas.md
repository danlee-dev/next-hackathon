# 01 · Strong B2B SaaS (예상: S/A)

## 세션 이름
```
LineupQ — 콜센터 음성 분석 SaaS Series A
```

## 발표 대본
```
안녕하세요, LineupQ 의 김민서입니다. 한국 콜센터 산업의 음성 데이터 50%가 분석 없이 폐기되고 있습니다.

저희가 풀려는 문제는 명확합니다. 한국 콜센터 시장 TAM 은 한국콘텐츠진흥원 2024 보고서 기준 4.2조 원, SAM 은 음성 분석 도입 가능 segment 1.1조 원, SOM 은 저희가 향후 3년 안에 잡을 수 있는 14% 의 1,540억 원입니다.

지금까지 성과는, ARR 24억 원, 18개월 누적 매출 38억 원, 월 churn 1.4%, gross margin 71% 입니다. 시리즈 A 시점 SaaS 평균 churn 이 3.2% 인 것 대비 절반입니다.

저희 차별점은 한국어 prosody-aware ASR. 영어 위주 모델은 한국어 콜에서 17% WER 인데, 저희는 4.8%. 이 차이로 KB 카드, NH 카드, 토스 콜센터에서 PoC 6 곳 모두 본계약 전환했습니다.

팀은 KAIST 음성공학 박사 2명, 엔지니어 11명. 시드 12억 받고 18개월 만에 PMF 달성, ARR 24억까지 왔습니다.

시리즈 A 30억 원으로 일본 진출 — Bizreach 와 Letter of Intent 체결 — 하반기 2026 KPI 는 ARR 90억, 일본 logo 5개입니다. 감사합니다.
```

## 심사 기준
```
1. 시장 규모와 SAM/SOM 의 정합성 (TAM 출처가 인용되는가)
2. Unit economics (CAC, LTV, gross margin)
3. Traction 구체성 (ARR, growth rate, churn)
4. 경쟁 우위의 측정 가능성
5. 다음 마일스톤의 검증 가능성
```

## 예상 결과
- trust_score: 80~92
- trust_grade: S 또는 A
- 김팩트: traction 인정 + 일본 진출 LoI 검증 요청
- 이공감: founder 침착함 + 숫자 신뢰
- 박독설: 추임새 거의 없음 — 톤만 약간 monotone 지적 가능
