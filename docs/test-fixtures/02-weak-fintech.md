# 02 · Weak Fintech (예상: C)

## 세션 이름
```
PayDream — 차세대 결제 플랫폼
```

## 발표 대본
```
안녕하세요, PayDream 의 박지호입니다. 저희는 음 그러니까 한국 핀테크 시장에 혁신적인 결제 솔루션을 제공하고 있습니다.

시장은 정말 큽니다. 한국 결제 시장은 어 사실 무한합니다. 모든 사람이 결제를 하기 때문이죠. 저희가 이 시장을 단연코 뒤집을 거라고 확신합니다.

저희 제품은 약간 그 카카오페이나 토스보다 훨씬 빠르고 안전합니다. 사용자들이 정말 좋아할 거예요. 어 그게 이제 저희가 만든 알고리즘이 정말 혁신적이거든요.

경쟁자는 사실 없다고 보시면 됩니다. 저희가 하는 방식은 아무도 못 하거든요. 음 그러니까 토스나 카카오는 그 옛날 방식이고요.

지금 베타 사용자가 약간 200명 정도 있는데, 음 곧 정말 많아질 거예요. 시리즈 A 50억 원 받아서 마케팅에 쓰려고 합니다. 감사합니다.
```

## 심사 기준
```
1. 시장 규모와 SAM/SOM 의 정합성 (TAM 출처 인용)
2. 비즈니스 모델 명확성
3. 차별점이 측정 가능한 지표로 표현되는가
4. Traction 의 구체성
5. 자금 사용처의 명확성
```

## 예상 결과
- trust_score: 35~50
- trust_grade: C 또는 F
- 김팩트: "결제 시장은 무한하다 → 시장 정의가 없음. 카카오페이/토스 시총·점유율 대비 위치도 모름."
- 이공감: "톤은 친근하지만 외운 멘트 같음. 자기 언어로 풀어내지 못함."
- 박독설: "'음', '그러니까', '약간', '어' 분당 12회 이상. '혁신적인' 3회, '단연코' 1회, '경쟁자 없음'으로 마무리. 인용·대체 제안 다수."
