# 03 · Medical AI (예상: B)

## 세션 이름
```
DiagnoSpark — 흉부 X-ray AI 판독 보조
```

## 발표 대본
```
안녕하세요, DiagnoSpark의 정수아입니다. 한국에서 매년 흉부 X-ray가 1,200만 건 촬영됩니다. 그런데 영상의학과 전문의는 1,800명. 1인당 6,600건씩 봐야 합니다. 결국 결핵·폐결절 같은 초기 신호가 늦게 잡힙니다.

저희는 이 상황에 AI 보조 판독 모델을 만듭니다. 학습 데이터는 분당서울대병원·서울아산병원과 IRB 통과 후 받은 익명화 영상 18만 건. 자체 검증에서 결핵 검출 민감도 92.1%, 폐결절 89.4% — 영상의학 전문의 평균에 근접합니다.

지금 단계는 식약처 의료기기 SaMD 등급 분류 진행 중이고요, 1차 임상 (서울아산병원) 6월 시작 예정입니다.

비즈니스 모델은 병원 구독료 + 판독당 과금 하이브리드. 한 가지 솔직하게 말씀드리면, 의료 AI 시장은 규제 통과 시점에 따라 매출 발생 시점이 달라집니다. 저희는 식약처 인허가 일정을 보수적으로 12개월로 잡고 있습니다.

팀은 영상의학과 펠로우 1명, ML 엔지니어 5명, 임상 시험 PM 1명입니다. 5억 원 시드로 시작해 모델·임상 1차까지 왔습니다.

시리즈 A 25억 원으로 식약처 승인까지 가고, 1년 안에 6개 종합병원 PoC 목표입니다. 감사합니다.
```

## 심사 기준
```
1. 의료 AI 특수성 (식약처 SaMD 인허가 경로 인지)
2. 학습 데이터의 출처와 IRB 통과 여부
3. 임상 검증 수치 (민감도/특이도)
4. 규제 리스크의 솔직성
5. Go-to-market 의 현실성 (병원 sales cycle)
```
