# 04 · Aggressive Edtech (예상: F)

## 세션 이름
```
LearnX — 인류 최고의 교육 플랫폼
```

## 발표 대본
```
안녕하세요. 저희는 LearnX 입니다. 음 사실 저희는 인류 역사상 가장 혁신적인 교육 플랫폼을 만들고 있고요, 어 그게 이제 단연코 최고의 솔루션입니다.

시장은 무한합니다. 모든 사람이 배우니까요. 한국 에듀테크 시장만 해도 어 정말 큽니다. 음 정확한 숫자는 저희가 아직 안 봤는데 그 매년 두 자릿수 성장하고 있다고 들었어요.

경쟁자는 절대적으로 없습니다. 클래스101? 그건 음 컨셉이 다르고요. 인프런? 그것도 사실 저희와 다른 카테고리입니다. 저희는 새로운 카테고리를 만들고 있어요.

저희 AI 가 어 그 학습자에게 맞는 콘텐츠를 추천하는데요, 음 ChatGPT 기반이라서 정말 똑똑합니다. 사용자들이 무조건 좋아할 수밖에 없습니다.

베타 사용자는 어 약간 50명? 이 정도 있는데, 음 그러니까 모두가 정말 만족하고 있어요. 곧 100만 명까지 갈 거라 보고요.

100억 원 받아서 마케팅에 다 쓰겠습니다. 감사합니다.
```

## 심사 기준
```
1. 시장 정의의 정확성 (TAM 출처와 한국 시장 정합성)
2. 차별점이 측정 가능한가
3. 경쟁사 분석의 정직성
4. Traction 의 구체성
5. 자금 사용처의 합리성
```

## 예상 결과
- trust_score: 25~40
- trust_grade: F (또는 매우 낮은 C)
- 김팩트: "'시장 무한', '한국 시장 큼' — 시장 분석이 없다. '경쟁자 절대적으로 없다' 는 시장 정의가 협소하다는 신호. 100억 원을 마케팅에 다 쓰는 건 PMF 미증명 단계에서는 자금 fire-out 신호."
- 이공감: "외운 멘트와 자기 말이 섞임. 자신감보다 과장이 앞선다. founder의 진정성이 검증되지 않았다."
- 박독설: "추임새 분당 15회 이상. '혁신적인'·'단연코'·'최고의'·'무조건'·'절대적으로' 5종 모두 등장. 인용 폭격 + 대체 제안 다수."

## 검증 포인트
- 백엔드 prompt가 컨텍스트 인지하면 *과장 표현 5종*을 정확히 인용하면서 지적해야 함.
- 만약 LLM이 "괜찮은 발표였지만 디테일이 부족..." 식 sycophantic 답변 내면 prompt가 약하다는 신호 — 즉시 페르소나 재튜닝.
